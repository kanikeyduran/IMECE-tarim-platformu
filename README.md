# İMECE - Tarım Arazisi Kiralama Platformu

Bu proje, atıl durumda olan tarım arazilerinin ekonomiye kazandırılması amacıyla geliştirilmiş web tabanlı bir pazar yeri (marketplace) uygulamasıdır. PHP ve MySQL kullanılarak geliştirilmiştir.

## Proje Hakkında
Arazi sahipleri, kullanmadıkları tarlalarını veya seralarını sisteme ekleyebilir; üreticiler ise bu ilanları inceleyerek "Gelir Paylaşımı" veya "Nakit Kira" modelleriyle teklif verebilirler.

## Kullanılan Teknolojiler
* **Backend:** PHP 8.2
* **Veritabanı:** MySQL (İlişkisel Veritabanı Mimarisi)
* **Frontend:** HTML5, CSS3
* **Sunucu:** Apache (XAMPP üzerinde geliştirildi)

## Kurulum
Bu projeyi kendi bilgisayarınızda çalıştırmak için:
1.  Bu repoyu indirin.
2.  `imece_db.sql` dosyasını phpMyAdmin üzerinden içe aktarın.
3.  `baglanti.php` dosyasındaki veritabanı ayarlarını yapılandırın.
4.  Tarayıcıdan `index.php` dosyasını çalıştırın.
